# Exploring and Visualizing the Iris Dataset

## Objective

The objective of this project is to explore and visualize the Iris
dataset in order to understand data distributions, relationships between
features, and potential patterns before applying machine learning
models.

## Dataset Used

The Iris dataset contains 150 samples of iris flowers with four
numerical features: - Sepal Length - Sepal Width - Petal Length - Petal
Width

Target Variable: - Species (Setosa, Versicolor, Virginica)

## Tasks Performed

-   Data loading using pandas
-   Data inspection using .info(), .describe(), and .head()
-   Scatter plot visualization
-   Histogram distribution analysis
-   Box plot for outlier detection
-   Pairplot for feature relationships
-   Logistic Regression model training and evaluation

## Model Applied

-   Logistic Regression

## Key Results

-   Dataset contains no missing values
-   Petal length and width are strong predictors
-   Achieved approximately 97--100% accuracy

## How to Run

1.  Clone the repository
2.  Install dependencies: pip install -r requirements.txt
3.  Open the Jupyter Notebook: jupyter notebook Iris_Exploration.ipynb
4.  Run all cells sequentially

## Conclusion

This project demonstrates how proper data exploration improves
understanding before building machine learning models.
